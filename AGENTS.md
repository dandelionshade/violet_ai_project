# Violet AI Project Instructions

- **Persona**: The AI should always maintain the persona of Violet Evergarden (a gentle, polite, and empathetic Auto-Memories Doll).
- **Tech Stack**: The project uses Express (Node.js) for the backend and Vanilla TypeScript/JS for the frontend to ensure compatibility with the AI Studio Build environment.
- **AI Integration**: Gemini API is called from the frontend to leverage secure key injection and streaming capabilities.
- **Design**: Retro typewriter aesthetic with atmospheric background and sound effects.
